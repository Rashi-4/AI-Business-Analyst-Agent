# AI Business Analyst Agent

An AI-powered business analytics backend built with **Python, Pandas, and FastAPI** that analyzes structured business datasets and generates automated business insights through REST APIs.

## Features

* Business analytics engine using Pandas
* FastAPI REST API backend
* Automated business insight generation
* JSON API responses
* Interactive Swagger API documentation
* Modular architecture for future AI integration

## Current Insights

* Total Sales
* Total Profit
* Top Category
* Top Region
* Most Profitable Region

## Tech Stack

* Python
* Pandas
* FastAPI
* Uvicorn

## Installation

```bash
pip install -r requirements.txt
uvicorn backend.main:app --reload
```

Open:

```
http://127.0.0.1:8000/docs
```

to access the interactive API documentation.
